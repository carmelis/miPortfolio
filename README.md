### Instalación

1. Clona el repositorio en tu local
   ```sh
   git clone https://github.com/carmelis/miPortfolio.git
   ```
2. Instala las dependencias NPM 
   ```sh
   npm install
   ```
3. Levanta la web
   ```sh
   npm start
   ```
