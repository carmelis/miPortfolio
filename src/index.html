Mis proyectos

Net Security

El proyecto realizado para Net Global Solutions consistia en una aplicacion de vigilancia para facilitar el monitoreo en las sucursales de sus clientes. Para llevarlo a cabo se crearon dos modulos, un modulo administrador construido con ReactJS y un modulo para cada vigilador que consistia en una aplicacion mobile creada con React Native. El backend de ambos sistemas se creo con ExpressJS.

Como en casa

El proyecto Como en Casa fue un ecommerce en el cual el usuario era capaz de visualizar los productos listados y agregarlos al carrito de compras. En el mismo se tuvo que coordinar el trabajo de 8 personas utilizando github como herramienta de versionado. Se utilizo ReactJS para el frontend y ExpressJS para el desarrollo del backend.
